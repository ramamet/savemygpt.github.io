SaveMyGPT Security & Privacy Notice

Last Updated: October 2025

1. No Cloud Storage

SaveMyGPT is a 100% local Chrome extension. Every conversation you capture stays in your browser via chrome.storage.local. We never upload or transmit your chats, prompt library selections, or settings to external servers.

2. No External Network Calls

The extension does not make any outbound API requests. SaveMyGPT interacts only with ChatGPT tabs and the packaged assets that ship with the extension.

5. Transparent Code

SaveMyGPT uses only standard Chrome APIs and inline scripts declared in the MV3 service worker, content script, popup, and options pages. There are no hidden trackers, analytics, or third-party libraries that transmit data.
